Current comparable claim:

WKVM is comparable enough to report same-shape native Gemma throughput against vLLM with an explicit no-HF guard, and the model/attention source boundaries now resemble the incumbent shape: model attention owns projection/norm/RoPE/output projection, while a backend-shaped path owns cache update/write and attention dispatch. It is not yet comparable as a full vLLM/SGLang-class inference engine because WKVM still lacks their generalized attention backend protocol, cache-spec/admission machinery, quantization/sharding/tensor parallel loader paths, and mature scheduler/runtime features.
